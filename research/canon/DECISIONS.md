# ArcLink CANON — Converged Operator Decision Brief

**What this is.** The single converged plan for every *deferred operator decision* surfaced by the 32-piece CANON whole-project dissection. Each piece's deferred call was first proposed by **Codex** (`research/canon/decisions/CANON-NN-*.codex.md`), then independently re-grounded against the live tree and **converged by Claude** (`research/canon/decisions/CANON-NN-*.decided.md`). Every plan is anchored to `docs/arclink/sovereign-control-node-symphony.md` (the controlling Symphony / North Star). This is the operator's decision brief: decisive where the two models agree, explicit where the operator must still pick.

**Method.** Codex proposed → Claude converged → both Symphony-anchored. Where Claude refined Codex, the refinement is code-grounded (re-opened the actual file/line, corrected stale premises, re-scoped severity/effort). Where a genuine policy/upgrade tradeoff remains, it is flagged as a **product fork** the operator owns.

**Totals.**
- **40 deferred decisions** across 32 pieces (several pieces resolved to *no operator decision*: CANON-01, 05, 14, 17, 32 are ratify-only; 06, 21, 23, 24, 29 are single-decision).
- **Verdicts:** ~15 *agree-codex* (ratify Codex as-is), ~25 *refine* (converge with code-grounded refinement), **0 unresolved model-vs-model disagreements**.
- **Genuine product forks the operator must pick: 6** (CANON-02, CANON-04 ×2 tuning, CANON-11, CANON-12, CANON-19 ×2). Marked **[FORK]** below.
- **Branch is NOT fully green:** 7 reds in `POST_CAMPAIGN_TEST_STATUS.md` (1 pre-existing, 2 env-suspect, **4 real cumulative cross-piece regressions**) gate the merge.
