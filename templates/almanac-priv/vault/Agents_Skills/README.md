# Agent Skills

This area holds shared skills and skill notes for agents across the organization.

Almanac seeds notes into subfolders:
- `Almanac/` for Almanac-managed skills installed by this repo
- `Hermes/` for official bundled skills from the pinned Hermes runtime

Expand them with:
- local examples
- rollout status
- owners
- caveats and troubleshooting
