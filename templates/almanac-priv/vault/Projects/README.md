# Projects

Use this area for project-shaped work when that helps an organization. It is a
convention, not a rule: project notes can also live in client, team, or flat
folders elsewhere in the vault and qmd will index them the same way.

What belongs here:
- project briefs
- decisions and milestones
- cross-functional coordination notes
- delivery and operational context worth retaining
